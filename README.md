# Super Store Sales Dashboard (Power BI)

## 📊 Project Overview

This project presents an interactive **Super Store Sales Dashboard** created using **Power BI**. The dashboard analyzes sales performance, profit trends, and customer purchasing behavior to help businesses make better data-driven decisions.

## 🚀 Key Metrics

* **Total Sales:** 522K
* **Total Quantity Sold:** 7K
* **Total Profit:** 68K
* **Average Delivery Time:** 7 Days

## 📈 Dashboard Insights

The dashboard provides multiple visual insights including:

* **Sales by Segment** (Consumer, Corporate, Home Office)
* **Sales by Payment Mode** (COD, Online, Cards)
* **Monthly Sales Trend**
* **Monthly Profit Analysis**
* **Sales by Category and Sub-Category**
* **Sales by Ship Mode**
* **State-wise Sales and Profit Map**
* **Top Performing States**
* **15-Day Sales Forecast**

## 🛠 Tools & Technologies

* Power BI
* Data Visualization
* Data Analysis
* Forecasting
